# 數辰 SPC 系統對接規範文件索引 (v1.1)

本目錄包含 SPC 系統整合之完整技術規格文件，旨在提供開發、測試與維護之標準指引。

## 📄 文件目錄

| 中文名稱 | 英文名稱 | 簡要說明 | 現狀 |
| :---- | :---- | :---- | :---- |
| **01 需求規格書** | Software Requirements Specification (SRS)  | 定義系統功能與非功能需求，是整個專案基礎 | 暫無 |
| **02 功能規格書** | Functional Specification Document (FSD) | 詳細描述系統功能與操作流程 | 暫無 |
| **03 系統設計文件** | System Design Document (SDD) | 系統架構、模組設計、技術選型 | 暫無 |
| **04 詳細設計文件**  | Detailed Design Document (DDD) | 模組內部邏輯與實作細節 | 暫無 |
| **05 資料字典** | Data Dictionary (DD) | 定義資料欄位、型態、長度與說明 |  |
| **06 資料流程圖** | Data Flow Diagram (DFD) | 描述資料在系統中的流動方式 |  |
| **07 實體關係圖** | Entity Relationship Diagram (ERD) | 資料庫結構與關聯設計 |  |
| **08 API 規格文件** | API Specification | 定義系統之間的介面與資料交換格式（常用 JSON） |  |
| **09 JSON 格式定義** | JSON Schema | 規範資料交換的 JSON 結構 |  |
| **10 技術提問回覆** | Frequently Asked Questions (FAQ) | 常見問題: 針對提問回覆 |  |
| **11 版次異動說明** |  | 以 JSON 範本呈現差異，差異處以註解標註 |  |
