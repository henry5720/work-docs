# 數辰 SPC 系統對接規範文件索引 (v1.3)

本目錄包含 SPC 系統整合之完整技術規格文件，旨在提供開發、測試與維護之標準指引。

## 📄 文件目錄

| 中文名稱 | 英文名稱 | 簡要說明 |
| :--- | :--- | :--- |
| [01 需求規格書](./01_SRS_需求規格書.md) | Software Requirements Specification (SRS) | 定義系統功能與非功能需求，是整個專案基礎 |
| [02 功能規格書](./02_FSD_功能規格書.md) | Functional Specification Document (FSD) | 詳細描述系統功能與操作流程 |
| [03 系統設計文件](./03_SDD_系統設計文件.md) | System Design Document (SDD) | 系統架構、模組設計、技術選型 |
| [04 詳細設計文件](./04_DDD_詳細設計文件.md) | Detailed Design Document (DDD) | 模組內部邏輯與實作細節 |
| [05 資料字典](./05_DD_資料字典.md) | Data Dictionary (DD) | 定義資料欄位、型態、長度與說明 |
| [06 資料流程圖](./06_DFD_資料流程圖.md) | Data Flow Diagram (DFD) | 描述資料在系統中的流動方式 |
| [07 實體關係圖](./07_ERD_實體關係圖.md) | Entity Relationship Diagram (ERD) | 資料庫結構與關聯設計 |
| [08 API 規格文件](./08_API_規格與用法說明.md) | API Specification | 定義系統之間的介面與資料交換格式（常用 JSON） |
| [09 JSON 格式定義](./09_JSON_格式規範.md) | JSON Schema | 規範資料交換的 JSON 結構 |
| [10 技術提問回覆](./10_FAQ_技術提問回覆.md) | Frequently Asked Questions (FAQ) | 常見問題：針對提問回覆 |
| [11 對接快速指南](./11_對接快速指南.md) | Integration Quickstart | 取得 Token → 提交匯入 → 輪詢，含 JS/Python 可執行範例 |
| [12 版次異動說明](./12_版次異動差異化說明.md) | Changelog | 文件集修訂歷史；以 JSON 範本呈現差異，差異處以註解標註 |
