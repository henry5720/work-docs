# 01 軟體需求規格書 (SRS)

## 1. 系統概述
本系統旨在提供統計製程管制 (SPC) 之自動化數據匯入與分析服務，支援大規模生產環境之資料對接。

## 2. 功能需求
### 2.1 資料匯入
- 支援單次上傳最高 10,000 筆之量測紀錄。
- 支援異步背景處理 (Asynchronous Background Processing) 模式。
### 2.2 資料追蹤
- 提供對應原始 Payload 之追蹤 ID (ref_id)，確保雙方對帳一致。

## 3. 非功能需求 (Non-functional Requirements)
### 3.1 效能指標 (Rate Limiting)
- **API 頻率限制**：單一租戶 (Tenant) 每分鐘上限 60 次請求。
- **背景任務佇列**：單一租戶最多併發處理 5 個匯入任務。
### 3.2 資料持久性與快取 (TTL)
- **任務狀態保存**：系統回傳之 Task ID 狀態於快取中保留 **24 小時**。
- **量測資料保存**：持久化於 MySQL 8.3，由資料庫管理策略維護。
