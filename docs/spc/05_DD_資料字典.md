# 05 資料字典 (DD)

> 回應提問 4：欄位規範與系統限制。

## 1. QuantitativeCCM (管制計畫表)

| 欄位名稱 (Key) | SQL 型態 | 長度限制 | 必填 | 說明 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | VARCHAR(36) | 36 | 是 | UUID v4 |
| `name` | VARCHAR(128) | 128 | 是 | 計畫名稱 (層別組合) |
| `part_number` | VARCHAR(128) | 128 | 是 | 產品料號 |
| `batch_number` | VARCHAR(128) | 128 | 是 | 產品批號 |
| `spec` | VARCHAR(128) | 128 | 否 | 規格描述 |
| `source` | VARCHAR(128) | 128 | 是 | 資料來源 (e.g., manual) |

## 2. QuantitativeCCMEntity (管制項目表)

| 欄位名稱 (Key) | SQL 型態 | 長度限制 | 必填 | 說明 |
| :--- | :--- | :--- | :--- | :--- |
| `characteristic_name`| VARCHAR(128) | 128 | 是 | 管制項目名稱 |
| `measurement_unit` | VARCHAR(128) | 128 | 是 | 單位 (e.g., mm, kg) |

## 3. QuantitativeCCMEntitySample (樣本資料表)

| 欄位名稱 (Key) | SQL 型態 | 長度限制 | 必填 | 說明 |
| :--- | :--- | :--- | :--- | :--- |
| `samples` | TEXT | - | 是 | 以逗號分隔之樣本字串 |
| `operator_name` | VARCHAR(64) | 64 | 是 | 操作人員名稱 |
| `idx` | INT | - | 是 | 系統流水號 |
