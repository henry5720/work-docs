### 數辰回覆

#### 備註

- 目前做法：採「扁平化統計」，不具備一一對應關係。
- 行為細節：
  - 匯入成功後，後端任務狀態會回傳 created_ccm_ids 與 created_entity_ids 列表。
  - 前端程式碼（ImportPagev2.jsx 第 367-371 行）僅將這些 ID 收集起來並計算 length（不重複總數）。
