## 🛠️ 如何快速「自動化」提取資訊？

身為前端工程師，您不需要「寫」作文，只需要從代碼中「複製並整理」：

1. **從 Swagger 提取 API (08, 09)**:
   - 直接參考 `/spc-backend/src/routers/private/quant_ccm/all_in_one.py`。
   - 關注 `@router.post("/all-in-one")` 及其 Payload 模型。

2. **從 SQLAlchemy Model 提取資料字典 (05)**:
   - 參考 `/spc-backend/src/database/models/quant_ccm.py`。
   - `Column(String(128), nullable=False)` $\rightarrow$ 代表 `VARCHAR(128)` 且 `必填`。

3. **從後端邏輯提取 DFD (06)**:
   - 觀察 `_run_all_in_one_import` 函數中的步驟：
     `接收 Payload` -> `寫入 Redis` -> `DB 檢核` -> `統計計算` -> `寫入 MySQL`。
