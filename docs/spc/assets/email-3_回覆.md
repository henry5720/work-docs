## 回覆稿

> 感謝比對。您對照的是較早版本的說明文件；這幾點在**最新版**都已補齊，主要集中在新增的**《11 對接快速指南》**：
>
> 1. **取得 Token 範例**：第二步提供完整 curl（端點 `/public/auth/signin`、`grant_type/username/password/client_id`）與回應格式。
> 2. **完整程式碼**：第三步提供 **JavaScript（Fetch）** 與 **Python（requests）** 可執行範例，含提交任務、輪詢（Polling）與 401 自動重新登入的錯誤處理。
> 3. **匯入介面與範本**：「網頁手動匯入」一節說明操作路徑與「下載範本」按鈕（範本由站台即時產生，網址依貴司部署而定，我們可另行提供）。另提醒：若貴司採**自寫腳本直接呼叫 API**，無需 CSV 範本，直接依 08/09 的 JSON 結構組裝即可。
> 4. **管制圖自動判定表**：見《08 API 規格》§3.10，依樣本數 n 自動判定（n=1→X̄-MR、2–10→X̄-R、n>10→X̄-S）。
> 5. **聯絡資訊**：文末技術支援信箱 `scaipm@shuchenai.com`。
>
