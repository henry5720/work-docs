# WebAR 體驗流程（現況）

> 給 PM 檢視用。可直接在下方調整順序、增刪內容,再交回工程調整。

## 流程

1. **首頁** — 點「WebAR 體驗」按鈕
2. **相機前導頁** — 提示「將瓶身 Logo 放入框內」、要求相機權限,點「啟動相機」
3. **掃描 Logo** — 對準瓶身 Logo 進行掃描
4. **開場影片** — 掃到 Logo 後自動播放產品介紹影片（3 秒後可 Skip）
5. **內容選單** — 出現 4 個按鈕:
   - **A 產品是什麼** → 播放單支影片
   - **B 產品特色** → 播放單支影片
   - **C 適用場景** → 播放單支影片
   - **D 調飲示範** → 調飲影片（向下滑動播放）+ 加入 LINE@
6. **加入 LINE@** — 於調飲頁引導加入,解鎖完整酒譜

## 一句話

首頁 → 相機前導 → 掃 Logo → 開場影片 → 選單（A/B/C 影片、D 調飲）→ 加入 LINE@
